프로젝트 개요

1. 이 프로젝트는 Python과 Discord API를 이용하여 소환사의 정보를 조회하고, 해당 정보를 디스코드 채팅방에 출력하는 봇을 만드는 것입니다.
2. 사용자가 봇에게 명령어를 입력하면, 봇은 해당 명령어를 수행하고 결과를 출력합니다.

개발환경

1. Python 3.9.7
2. Discord.py 1.7.3
3. RiotWatcher 4.18.1

구현기능

1. !소환사추가 [소환사 이름]: 소환사를 추가합니다.
2. !소환사삭제 [소환사 이름]: 소환사를 삭제합니다.
3. !확인하기: 추가된 소환사의 정보를 출력합니다.
4. 소환사 추가 및 삭제 기능은 데이터베이스에 소환사 정보를 저장합니다.
5. 봇이 다수의 채널에서 사용될 수 있도록, 각 채널별로 소환사 리스트를 관리합니다.
6. API 요청 제한 시간을 고려하여 일정 시간마다 API 요청을 보내기 위해 비동기 방식으로 구현하였습니다.

한계점

1. 이 봇은 오직 한 서버에서만 사용 가능합니다. 다른 서버에서는 소환사 정보를 볼 수 없습니다.
2. 소환사 정보가 업데이트되는 시간에 따라 봇이 특정 시간 이후에는 업데이트된 정보를 제공하지 못할 수 있습니다.
3. 봇의 소환사 정보 조회 기능은 높은 수준의 정확도를 보장하지 못합니다.

보완점

1. 봇의 기능을 확장하여, 소환사의 정보뿐만 아니라, 게임 내 정보도 조회할 수 있도록 구현해보는 것이 좋을 것입니다.
2. 데이터베이스를 사용하여 소환사 정보를 저장하는 것이 아니라, 다른 방식으로 소환사 정보를 저장하여 보다 빠르게 정보를 제공할 수 있도록 개선할 수 있습니다.

결론

1. 이 프로젝트를 통해 Python과 Discord API, RiotWatcher를 이용하여 봇을 개발하는 기술을 습득하였습니다.
2. 해당 봇은 한 서버에서 소환사의 정보를 조회하고 출력하는 기능을 제공합니다.
3. 앞으로 보완 및 개선을 통해 더욱 다양한 기능을 제공할 수 있는 봇으로 발전시켜나가는 것이 중요합니다.
